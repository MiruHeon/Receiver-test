# UDP 기반 패킷 손실·지연 통신 테스트 시스템 (Receiver-Test)

Python의 socket 라이브러리를 사용하여 제작된 UDP 통신 테스트 시스템입니다.
송신기에서 전송되는 패킷을 수신하며, 패킷 손실과 지연을 시뮬레이션하여 네트워크 불안정 상황을 실험할 수 있습니다.

## 주요 기능

UDP 기반 패킷 수신
송신 측 패킷 손실(loss) 시뮬레이션
지연(delay) 적용 가능 (밀리초 단위)
수신 패킷 실시간 출력 및 송신자 정보 확인 가능

## 기술적 특징

socket 라이브러리를 사용했습니다. (IPv4, UDP)
직접 만든 channel.transmit() 함수를 통해 패킷 손실률과 지연 시간 설정 가능합니다.
UDP의 특성상 순서 보장 X / 재전송 X 이기 때문에 현실 네트워크의 불완전성을 학습 가능합니다.
